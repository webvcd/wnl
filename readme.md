# 中国农历公历换算算法

## 参考标准

算法根据《中华人民共和国国家标准GB/T33661—2017〈农历的编算和颁行〉》标准开发，明确了干支纪年和生肖纪年起于正月初一0点，与农历新年同步。

## 方法调用

```js
// 农历日期: 2022年(大致)闰四月初十
calendar.getDateByLunar(2022,4,10,true);

// 公历日期：2022年10月1日
calendar.getDateBySolar(2022,10,1);

// 今天
calendar.getToday();
```
